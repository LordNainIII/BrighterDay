# Brighter Day

Brighter Day is an AI-assisted therapist support platform designed to streamline session documentation and reflective clinical review.

Built with React, Vite, Firebase, and OpenAI, the platform aims to reduce administrative burden while supporting evidence-informed therapeutic reflection.

## Core Features

- Secure therapist authentication and account management  
- Client profile creation and management  
- Session recording and audio upload  
- Automatic transcription pipeline via OpenAI Whisper  
- AI-generated session summaries  
- Interactive AI chat against transcript and clinical knowledge base  
- Firebase cloud storage and Firestore integration  

## Purpose

Brighter Day is intended as a therapist support tool only and does not replace professional judgement, diagnosis, or clinical expertise.
